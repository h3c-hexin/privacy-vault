### 1.2 每个页面的核心功能点

| 页面 | 核心功能 |
|------|----------|
| 计算器伪装首页 | 完整科学计算器功能；输入密码序列（如 `1234=`）后切换到保险箱 |
| 首次设置页 | 设置 4-6 位 PIN；注册指纹/面部识别；安全问题设置；简要功能引导 |
| 解锁页 | PIN 输入；指纹/面部识别触发；错误次数记录与锁定 |
| 主页 | 文件夹/相册网格展示；按类型筛选（全部/图片/视频）；快速导入入口；存储用量展示 |
| 文件夹详情页 | 缩略图网格；多选模式；排序切换；批量操作工具栏 |
| 图片预览页 | 全屏解密预览；双指缩放；左右滑动切换；分享/导出/删除操作 |
| 视频播放页 | 全屏流式解密播放；播放/暂停；进度拖拽；音量/亮度手势 |
| 文件导入页 | 系统相册/文件管理器选择器；批量勾选；导入后删除原文件选项 |
| 回收站页 | 已删除文件列表；恢复操作；彻底删除；清空回收站 |
| 设置页 | 密码管理入口；自动锁定时间；导入后删除开关；存储统计；支持开发者 |
| 密码修改页 | 验证旧 PIN；设置新 PIN；重新绑定生物识别 |

## 3. 页面详细说明

### 3.1 计算器伪装首页

**用途**：App 以计算器身份出现在桌面，用户输入特定密码序列后进入保险箱。

**UI 元素**：

| 区域 | 元素 | 说明 |
|------|------|------|
| 顶部 | 计算器显示屏 | 显示输入的数字和计算结果，与系统计算器风格一致 |
| 中部 | 数字键盘 0-9 | 标准计算器布局 |
| 中部 | 运算符 +, -, x, / | 基本四则运算 |
| 中部 | 功能键 C, %, +/-, . | 清除、百分比、正负号、小数点 |
| 底部 | = 号按钮 | 计算结果 / 触发密码验证 |

**用户操作**：
- 正常使用计算器进行运算（真实可用）
- 输入 PIN 序列后按 `=`，若密码正确则过渡进入保险箱
- 密码错误时显示正常计算结果，不暴露任何异常

**跳转逻辑**：
- 密码正确 --> 过渡动画 --> 主页
- 密码错误 --> 显示计算结果，停留在计算器

**备注**：可在设置中开启/关闭伪装入口。首次设置时引导用户选择是否启用。

---

### 3.2 首次设置页

**用途**：首次安装后引导用户完成安全配置。

**UI 元素**：

| 区域 | 元素 | 说明 |
|------|------|------|
| 顶部 | 步骤指示器 | 当前步骤 / 总步骤（如 1/4） |
| 顶部 | App Logo + 标语 | "你的私密空间" |
| 内容区 Step1 | PIN 输入框 (4-6位) | 数字键盘，密码不可见，圆点显示 |
| 内容区 Step1 | 确认 PIN 输入框 | 二次输入确认 |
| 内容区 Step2 | 指纹/面部识别设置 | 调用系统生物识别注册 |
| 内容区 Step3 | 安全问题选择 | 下拉选择问题 + 输入答案 |
| 内容区 Step4 | 功能引导卡片 (3张) | 滑动浏览核心功能介绍 |
| 底部 | [下一步] / [跳过] / [完成] | 步骤导航按钮 |

**用户操作**：
- 设置 PIN（必须）
- 绑定生物识别（可跳过）
- 设置安全问题（必须）
- 浏览功能引导（可跳过）

**跳转逻辑**：
- 完成所有步骤 --> 主页（空状态）
- 中途退出 App --> 下次启动继续未完成步骤

---

### 3.3 生物识别/PIN 解锁页

**用途**：日常打开 App 或从后台恢复时的身份验证。

**UI 元素**：

| 区域 | 元素 | 说明 |
|------|------|------|
| 顶部 | App Logo / 锁图标 | 品牌识别 |
| 中部 | PIN 输入圆点 (4-6个) | 随输入填充 |
| 中部 | 数字键盘 0-9 | 底部布局，无字母 |
| 中部 | 删除键 | 回退一位 |
| 底部 | "忘记密码？" | 跳转安全问题重置流程 |
| 浮层 | 指纹识别弹窗 | 自动弹出（如已绑定） |

**用户操作**：
- 输入 PIN 解锁
- 使用指纹/面部识别解锁
- 点击"忘记密码"重置

**跳转逻辑**：
- 验证成功 --> 主页
- 验证失败 --> 震动反馈 + 错误提示 + 错误计数
- 忘记密码 --> 安全问题验证 --> 重设 PIN

**备注**：连续错误 >= 3 次触发入侵者拍照（后台静默），可在设置中开启/关闭。

### 3.4 主页（文件总览）

**用途**：用户进入保险箱后的主界面，展示所有文件夹和快捷操作。

**UI 元素**：

| 区域 | 元素 | 说明 |
|------|------|------|
| 顶部导航栏 | "保险箱" 标题 | 左侧 |
| 顶部导航栏 | 搜索图标 | 右侧，点击展开搜索栏 |
| 顶部导航栏 | 更多菜单 (...) | 右侧，包含回收站入口、设置入口 |
| Tab 栏 | [全部] [图片] [视频] | 按文件类型筛选 |
| 内容区 | 文件夹/相册网格 | 2 列网格，每个卡片显示封面缩略图 + 名称 + 文件数 |
| 内容区 (空状态) | 引导插画 + 文字 | "还没有文件，点击下方按钮开始导入" |
| 底部 | 存储用量条 | 已用空间 / 设备可用空间 |
| 右下角 FAB | "+" 浮动按钮 | 导入文件入口 |

**用户操作**：
- 点击文件夹 --> 进入文件夹详情页
- 点击 "+" --> 进入文件导入页
- 长按文件夹 --> 显示操作菜单（重命名/删除/移动）
- 切换 Tab 筛选文件类型
- 点击搜索 --> 展开搜索栏
- 点击更多菜单 --> 回收站 / 设置

**跳转逻辑**：
- 文件夹卡片 --> 文件夹详情页
- "+" 按钮 --> 文件导入页
- 搜索 --> 搜索结果页（内嵌）
- 更多 > 回收站 --> 回收站页
- 更多 > 设置 --> 设置页

### 3.5 相册/文件夹详情页

**用途**：展示某个文件夹内的所有文件，支持浏览和管理。

**UI 元素**：

| 区域 | 元素 | 说明 |
|------|------|------|
| 顶部导航栏 | 返回箭头 | 返回主页 |
| 顶部导航栏 | 文件夹名称 + 文件数 | 如 "私密相册 (42)" |
| 顶部导航栏 | 排序图标 | 按日期/名称/大小排序 |
| 顶部导航栏 | 视图切换 | 网格/列表切换 |
| 内容区 | 缩略图网格 | 3-4 列网格，图片显示缩略图，视频显示缩略图+时长角标 |
| 内容区 (空状态) | "暂无文件" + 导入按钮 | 引导用户导入 |
| 右下角 FAB | "+" 按钮 | 向此文件夹导入文件 |
| 底部 (多选模式) | 批量操作栏 | [移动] [删除] [导出] [分享] |

**用户操作**：
- 点击文件 --> 图片预览 / 视频播放
- 长按文件 --> 进入多选模式
- 点击排序 --> 切换排序方式

### 4.5 动画和转场建议

| 场景 | 动画类型 | 时长 | 说明 |
|------|---------|------|------|
| 计算器 --> 主页 | 形变过渡 | 400ms | 计算器界面缩小并淡出，主页从中心展开 |
| 页面间导航 | 共享元素过渡 | 300ms | 缩略图放大为全屏预览（Material Hero） |
| 解锁成功 | 锁图标动画 | 300ms | 锁打开 --> 淡出 --> 主页淡入 |
| 文件删除 | 卡片收缩消失 | 200ms | 被删项缩小并淡出，其余项平滑填补 |
| 多选模式激活 | 弹簧效果 | 200ms | 所有卡片轻微缩小，勾选框弹出 |
| 底部弹出表 | 滑入/滑出 | 250ms | 从底部滑入，背景半透明遮罩淡入 |
| 导入进度 | 进度条填充 | 连续 | 平滑进度增长，非跳跃 |
| Tab 切换 | 交叉淡入淡出 | 200ms | 内容区域平滑切换 |
| 文件夹创建 | 缩放弹入 | 250ms | 新卡片从缩放 0 弹入到正常大小 |
| Toast/Snackbar | 底部滑入 | 200ms | 从底部滑入，自动滑出消失 |

**动画原则**：
- 所有动画使用缓动曲线（easeInOut），避免线性动画
- 动画时长控制在 200-400ms，避免拖沓
- 提供 "减少动画" 无障碍选项，尊重系统设置
- 加载相关动画不设上限时长，跟随实际进度

## 5. 导航结构

### 5.1 主导航方式

**采用方案：顶部导航栏 + FAB + 右上角菜单**

不使用底部 Tab Bar，原因：
- 保险箱 App 的核心操作路径是 "进入 --> 浏览文件 --> 操作 --> 退出"，是垂直纵深型而非横向平铺型
- 页面数量有限（11 个），不需要频繁在平级页面间切换
- 底部空间留给 FAB（导入按钮）和批量操作栏更有价值
- 简洁的导航减少界面复杂度，符合隐私工具的"安静"气质

**导航元素分布**：

```
+----------------------------------------------+
|  [<-]  保险箱         [搜索] [更多菜单 ...]    |  <-- 顶部导航栏
+----------------------------------------------+
|  [全部]  [图片]  [视频]                        |  <-- Tab 栏（内容筛选）
+----------------------------------------------+
|                                              |
|           文件夹/文件网格内容区                  |
|                                              |
|                                              |
|                                              |
|                                      [+]     |  <-- FAB 浮动按钮
+----------------------------------------------+
```

**更多菜单展开内容**：
- 回收站
- 新建文件夹
- 设置

### 5.2 页面返回逻辑

| 当前页面 | 返回目标 | 返回方式 |
|---------|---------|---------|
| 文件夹详情页 | 主页 | 返回箭头 / 系统返回键 |
| 图片预览页 | 文件夹详情页 | 返回箭头 / 下滑手势 / 系统返回键 |
| 视频播放页 | 文件夹详情页 | 返回箭头 / 系统返回键 |
| 文件导入页 | 上一页面 | 关闭按钮 / 系统返回键 |
| 回收站页 | 主页 | 返回箭头 / 系统返回键 |
| 设置页 | 主页 | 返回箭头 / 系统返回键 |
| 密码修改页 | 设置页 | 返回箭头 / 系统返回键 |
| 支持开发者页 | 关于页 | 返回箭头 / 系统返回键 |
| 主页 | 锁定并退出 App | 系统返回键（弹出确认或直接锁定） |

**特殊返回逻辑**：
- 多选模式下按返回键：先退出多选模式，再按一次才返回上级
- 搜索模式下按返回键：先关闭搜索栏，再按一次才返回
- 导入进行中按返回键：最小化导入进度（后台继续），不中断导入
- 主页按返回键：锁定 App 并退到后台（不弹确认对话框，保持安全习惯）

### 5.3 深度链接考虑

由于隐私保险箱的特殊性质，**不支持系统级深度链接**（Deep Link），原因：
- 深度链接会暴露 App 的真实身份
- 从通知或其他 App 跳转会绕过解锁流程
- 不符合隐私保护的设计原则

**App 内部导航恢复策略**：
- App 从后台恢复时，始终先展示解锁页
- 解锁成功后恢复到上次退出时的页面状态
- 使用内存中的导航栈恢复，不持久化到磁盘（防止信息泄露）
