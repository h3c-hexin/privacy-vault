## 0. 商业模式

### 模式：全功能免费 + 赞赏码捐赠

**所有功能完全免费**，无付费墙、无广告、无文件数量限制。

通过"设置 → 关于 → 支持开发者"页面引导自愿捐赠：
- 微信赞赏码（图片展示，用户长按保存后扫码）
- 金额随意，完全自愿
- 赞赏码需提前设置自定义昵称，避免泄露真实姓名

### 选择此模式的原因
- 零门槛：不需要企业资质、不需要接入支付 SDK
- 用户体验最佳：所有功能开箱即用，无中断
- 信任感强：完全免费 + 无广告在隐私类 App 中是差异化优势
- 开发简单：无需处理内购逻辑、商品管理、支付回调

### 注意事项
- 收入预期极低，以兴趣驱动和用户口碑为主要目标
- 赞赏码图片需内置到 App 资源中
- 后续如需商业化，可随时切换到内购模式

> 隐私保险箱让你用最简单的方式，把手机里最私密的照片、视频和文件锁进一个只有你能打开的加密空间。计算器伪装入口让任何人都看不出这是一个保险箱，AES-256 军事级加密确保即使手机丢失，数据也无法被读取。

## 3. 用户故事（MVP 核心）

### US-01: 首次设置保险箱
**作为** 一个刚安装 App 的用户，**我想要** 快速设置 PIN 密码并绑定指纹，**以便** 之后能安全又便捷地进入私密空间。

### US-02: 通过计算器伪装入口进入
**作为** 不希望别人发现隐私 App 的用户，**我想要** 桌面上只看到"计算器"App，输入密码才进入保险箱，**以便** 即使别人翻看手机也不会发现。

### US-03: 批量导入照片并自动删除原文件
**作为** 想要隐藏大量私密照片的用户，**我想要** 一次性多选导入并自动删除原图，**以便** 一步到位完成转移，不留痕迹。

### US-04: 浏览和播放加密视频
**作为** 在保险箱中存储了视频的用户，**我想要** 在 App 内直接流畅播放，**以便** 不需要先解密导出就能观看。

### US-05: 用指纹快速解锁
**作为** 频繁使用保险箱的用户，**我想要** 指纹一触即开，**以便** 不用每次都输入 PIN 码。

### US-06: 整理文件到不同相册
**作为** 存储了大量文件的用户，**我想要** 创建不同相册/文件夹分类管理，**以便** 快速找到需要的内容。

### US-07: 从回收站恢复误删文件
**作为** 不小心删除了文件的用户，**我想要** 在回收站中恢复，**以便** 重要文件不会因误操作永久丢失。

### US-08: 导出文件分享给信任的人
**作为** 需要分享某些文件的用户，**我想要** 将选定文件解密导出或直接分享，**以便** 方便地发送给对方。

5. 纯本地存储 — 隐私零泄露

## 5. 风险与注意事项

### 国内市场特殊考量
- **应用商店审核**：伪装类 App 审核趋严 → 强调"隐私保护"而非"隐藏"，伪装功能默认不开启
- **PIPL 合规**：仅申请必要权限，编写完整隐私政策，不收集用户行为数据
- **数据安全法**：使用标准公开算法，声明仅加密本地文件
- **应用备案**：预留 2-4 周备案审核时间
- **未成年人保护**：强调商务/个人隐私保护场景

### 技术风险
- 大文件加密性能 → 分块加密 + 流式解密
- 密钥管理安全 → Android Keystore + Argon2id
- Android 碎片化 → 最低 Android 8.0，主流机型兼容测试
- 存储权限变更 → Android 13+ 使用 MediaStore API
- 用户忘记密码 → 安全问题 + 风险告知

### 用户体验风险
- 导入速度慢 → 后台加密 + 进度展示
- 计算器伪装不够真实 → 实现完整科学计算器
- 存储空间焦虑 → 设置中展示占用统计
- 误操作数据丢失 → 30 天回收站 + 二次确认
