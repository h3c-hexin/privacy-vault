## 第一条 服务说明

1.1 本应用是一款纯本地化的加密文件管理工具，主要功能包括：

- 将用户选择的图片、视频等文件通过 AES-256-GCM 算法加密后存储在设备本地
- 在应用内预览、管理加密文件
- 将加密文件解密导出到设备存储
- 通过 PIN 码、生物识别等方式保护应用入口
- 计算器伪装入口，保护应用隐私

1.2 本应用完全在您的设备本地运行，不依赖网络连接，不提供任何在线服务或云存储功能。

1.3 本应用完全免费，无广告、无内购。"设置 - 关于 - 支持开发者"页面中的赞赏功能为纯自愿行为，不影响任何功能的使用。

## 第二条 账号与密码

2.1 首次使用本应用时，您需要设置一个 4-6 位 PIN 码作为主访问凭证。该 PIN 码将用于派生文件加密密钥。

2.2 您可以选择启用生物识别（指纹/面部识别）作为快捷解锁方式。生物识别解锁依赖您设备的硬件能力，开发者不保证所有设备均可支持。

**2.3 关于 PIN 码的重要警告（请务必仔细阅读）：**

- **您的 PIN 码是访问加密数据的唯一凭证**
- **开发者不存储、不记录、不传输您的 PIN 码**
- **开发者无法通过任何技术手段帮您找回或重置 PIN 码**
- **如果您忘记 PIN 码，您存储在本应用中的所有加密数据将永久无法访问，且无法恢复**
- **开发者对因 PIN 码遗忘导致的数据无法访问不承担任何责任**

2.4 您有义务妥善保管自己的 PIN 码，不要将其告知他人。因您主动或被动泄露 PIN 码导致的任何后果，由您自行承担。

## 第四条 数据安全与风险提示

**4.1 数据不可恢复风险**

**您充分理解并同意，在以下情形中，您存储在本应用中的加密数据将永久丢失且无法恢复，开发者对此不承担任何责任：**

- **忘记 PIN 码**
- **卸载本应用**

- **恢复设备出厂设置**
- **设备损坏、丢失或被盗**
- **使用紧急数据销毁功能**
- **Android 系统升级导致应用数据清除**
- **任何其他导致应用数据目录被删除或损坏的情形**

**4.2 数据备份责任**

**开发者强烈建议您定期将重要文件导出并备份到安全的外部存储中。本应用不提供云端备份功能，所有数据仅存在于您的设备本地。您自行决定是否进行备份以及备份的频率和方式，开发者不对因未备份导致的数据丢失承担责任。**

**4.2.1 版本升级风险**

**本应用可能通过版本更新改进加密算法、存储格式或安全机制。尽管我们会尽力确保向后兼容性，但因软件升级导致的数据格式变更、解密异常等情况，开发者不承担数据丢失或不可访问的责任。建议您在每次更新应用前，先将重要文件导出备份。**

4.3 加密安全性声明

本应用采用 AES-256-GCM 加密算法和 Argon2id 密钥派生算法，均为国际公认的安全加密标准。但开发者不对加密算法的绝对安全性作出保证，密码学领域可能出现的新的攻击方法或计算能力的突破可能影响加密安全性。
